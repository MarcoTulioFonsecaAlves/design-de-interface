Em HTML, CSS e JS, crie uma página que apresente uma breve explicação sobre o curso de design. A página deve possuir um link para uma segunda página que apresente um resumo de 2 unidades curriculares do curso.

RESPOSTA

O curso de design é uma formação educacional que prepara os estudantes para trabalhar na criação de soluções visuais, funcionais e estéticas para uma variedade de produtos, comunicações e ambientes. O design envolve a combinação de elementos como forma, cor, tipografia, espaço e imagens para transmitir uma mensagem ou cumprir um propósito específico. Ele abrange diversas áreas e disciplinas, incluindo design gráfico, design de produto, design de interiores, design de moda, design de comunicação, design de experiência do usuário (UX) e design de interface do usuário (UI), entre outras.
